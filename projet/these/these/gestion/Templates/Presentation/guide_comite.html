Étape 2: Évaluation de la Copie 0

Examen initial du projet de recherche

La première phase d'évaluation consiste à examiner la copie 0 soumise par le doctorant. Ce document présente le projet de recherche initial.

Pour accéder aux copies 0 à évaluer :

  1. Dans votre page copie 0, une liste de thése est déja dresser
  2. Filtrez par "Copie 0" pour voir uniquement les documents initiaux
  3. Cliquez sur le document pour le télécharger et verifier le plagiat

Pour soumettre votre évaluation :

  1. Choisissez l'une des décisions suivantes :
    • Approuvé
    • réjetté
Conseil : Soyez précis dans vos commentaires pour permettre au doctorant d'améliorer son projet de recherche.

Étape 3: Verification de plagiat

Analyse des avancées de la recherche

Cette étape permet d’analyser le rapport intermédiaire afin de détecter d’éventuelles similarités avec d’autres documents académiques enregistrés dans notre base de données. Elle vise à garantir l’originalité du travail de recherche du doctorant et à prévenir tout risque de plagiat.

Étape 4: Composition du Jury

Constitution du jury de soutenance

À cette étape, les membres du jury de soutenance sont désignés. Le jury est composé d’enseignants-chercheurs qualifiés chargés d’évaluer la qualité scientifique du travail présenté par le doctorant lors de la soutenance finale.
